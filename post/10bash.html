10 Command Bash Linux yang Sering Digunakan

ditulis oleh IPTEK FORKOM TRI 2021 pada 20 Mei 2021.

  1. pwd

    2. Melihat pada direktori mana kita sedang bekerja.

  2. ls

    3. Command 'ls' digunakan untuk melihat isi direktori yang saat ini sedang diakses, jika ingin melihat direktori lain maka command ls disusul dengan path direktori yang akan dilihat.

  3. mkdir

    4. Perintah untuk membuat folder/direktori.

  4. rm

    5. Perintah untuk menghapus file.

  5. mv

    6. Perintah untuk memindahkan suatu file/direktori, bisa juga digunakan untuk mengubah nama file/direktori.

  6. nano

    7. Command ini digunakan untuk menggunakan text editor nano, nano merupakan salah satu text editor bawaan yang sering digunakan untuk menyunting file konfigurasi linux.

  7. Sudo

    8. Sudo merupakan singkatan dari “SuperUser Do” dan berfungsi untuk menjalankan task yang memerlukan hak akses (permission) administrative atau root.

  8. touch

    9. Touch adalah perintah dasar Linux yang memperbolehkan Anda membuat file baru yang kosong melalui baris perintah Linux.

  9. find

    10. Find adalah perintah dasar Linux yang ditujukan untuk mencari suatu file dengan direktori tertentu. Umumnya digunakan untuk memeriksa apakah dalam suatu direktori terdapat file yang dicari atau tidak.

  10. ssh-copy-id

    11. Menyalin SSH public key kita ke remote server untuk keperluan login SSH menggunakan metode public key.
